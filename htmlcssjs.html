HTML/CSS/JS project

Project Title: Creating a Portfolio using HTML

Project Description: The project aims to create a website that includes seven sections to apply HTML, CSS, and JavaScript. The objective of this project was to create a website that contains seven sections, including a navigation bar with four sections. The first two sections contain text content, and the third section includes a carousel design. The fourth section is a responsive webpage using CSS Flex and Grid Layout with CSS Declarations. The fifth section is designed as a carousel with Flex and Grid Layout as well. The sixth and seventh sections contain temperature conversion that pops up in windows using JavaScript Functions and Document Object Model.

Project Type: Individual Project
